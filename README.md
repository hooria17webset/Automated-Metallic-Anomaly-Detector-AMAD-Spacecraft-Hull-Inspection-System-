# Automated-Metallic-Anomaly-Detector-AMAD-Spacecraft-Hull-Inspection-System-
Built a two-stage ML pipeline (Anomalib + YOLO) for real-time spacecraft hull defect detection, quantized for edge deployment. Achieved 9.28 FPS inference across 415 frames, detecting 99 structural faults (23.86% rate) while cutting classifier compute load by 76% via a cascaded gatekeeper design.
